# 🛡️ LostCryptoHelp Starter

Full-stack crypto protection toolkit focused on wallet safety, token approval revokes, and real-time scan alerts.

---

## ⚙️ Features
- 🔐 RevokeShield — token approval scanner & revoker
- 🔎 Chain support for Ethereum & BSC
- 🧠 Dynamic spender/token detection from on-chain data
- 🔄 Metamask integration for 1-click revoke
- 💾 Logs stored in `revoke_logs.json` locally
- 📦 Built with Flask + React (easy to deploy)
